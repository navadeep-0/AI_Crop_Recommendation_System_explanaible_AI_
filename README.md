# AI_Crop_Recommendation_System_explanaible_AI_

AI-powered crop recommendation system using ensemble machine learning, SHAP explainability, groundwater sustainability analysis, and market intelligence for sustainable agriculture. Using the official data collected from the government of telangana

## Overview
An AI-powered crop recommendation system using ensemble machine learning, SHAP explainability, groundwater sustainability analysis, and market intelligence for sustainable agriculture.

## Features
- Crop prediction using ensemble learning
- Random Forest, Gradient Boosting, XGBoost
- SHAP-based explainable AI
- Groundwater sustainability analysis
- Eco-Economic Index (EEI)
- Multilingual reporting
- Market price analysis

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Matplotlib

## Project Workflow
1. Data preprocessing
2. Feature engineering
3. Ensemble model training
4. SHAP explainability
5. Sustainability analysis
6. Economic analysis
7. Final crop recommendation

## Results
- Achieved over 99% prediction accuracy
- Improved sustainable crop planning
- Integrated explainable AI for transparency
